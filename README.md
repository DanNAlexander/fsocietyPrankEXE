# fsocietyPrankEXE

A harmless Mr. Robot-themed prank app for Windows. Runs a fake "hacking" sequence — fullscreen terminal aesthetic, animated progress bar, blinking ransomware warning — then reveals it was just a prank.

Nothing is modified, encrypted, or deleted. This is purely visual.

What it does
Displays the fsociety ASCII logo with a green flicker effect
Counts down with a warning message
Scrolls through fake "encrypting files..." status lines with a progress bar
Flashes a blinking YOUR FILES HAVE BEEN ENCRYPTED alert
Reveals the joke — Hello, friend. Nothing has been modified.
Press Escape to close after the reveal
Requirements
Python 3.x
tkinter (included with standard Python on Windows)
Run

python fsociety.py
Or use the prebuilt fsocietyexe/fsociety.exe — no Python install needed.
